# AI Ecotourism Assistant

Dự án chatbot du lịch Núi Bà Đen với frontend React và backend FastAPI.
Ứng dụng cho phép quét QR, chọn địa điểm, xem giới thiệu và hỏi AI Guide.
Dữ liệu du lịch được đọc từ file Excel và ảnh địa điểm được phục vụ từ thư mục local.
Backend hỗ trợ RAG bằng ChromaDB và kết nối LLM qua OpenAI Python SDK.
Mục tiêu là tạo trải nghiệm tham quan thông minh, gọn nhẹ và dễ demo.
